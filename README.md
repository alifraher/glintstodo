# TO DO LIST APP

## Description

To Do List App is a simple todo application for everyone.

You can [access the demo here](https://www.figma.com/proto/C8ayKd2eiI8gRYFga2NVRciK/Todo-List-v2?node-id=4%3A0&scaling=scale-down)

## Features

* Create todo
* Read todo
* Update todo
* Delete todo
* Check complete todo
